# dot-council

dot-council is an automated pixel-art content publishing tool. A multi-agent AI pipeline (six AI agents) collaborates daily to generate RPG-style battle videos and pixel character illustrations, which are posted automatically to social media platforms.

This site hosts the legal documentation for the project.
